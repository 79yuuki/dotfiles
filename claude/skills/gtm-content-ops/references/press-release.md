# プレスリリース運用

## 「2年分を3日で」パターン

プレスリリースが出せない最大の理由は「ネタがない」ではなく「手が回らない」。
仕組みを整えれば、過去にやったことを一気にプレス化できる。

## プレスリリースになるネタの棚卸し

以下のイベントは全てプレスリリースの対象:

| カテゴリ | 具体例 |
|---|---|
| プロダクト | 新サービスリリース、大型アップデート、新機能追加 |
| パートナーシップ | 業務提携、技術提携、共同開発 |
| 人事・組織 | 新メンバー参画、アドバイザー就任、組織変更 |
| イベント | 登壇、主催イベント、出展 |
| アンバサダー | 公式アンバサダー就任、認定取得 |
| 実績 | ユーザー数達成、売上マイルストーン、受賞 |
| 調査・レポート | 業界調査結果、ホワイトペーパー公開 |
| 資金調達 | 資金調達完了、助成金採択 |
| 研修・教育 | 研修実施、講座開設、書籍出版 |
| メディア | メディア掲載（掲載報告PR）、インタビュー |

### 棚卸しワークフロー

```
1. 直近6-12ヶ月のイベントをリスト化
   - カレンダー / SNS投稿 / 議事録から抽出
2. 上記カテゴリに分類
3. 各イベントの関係者を特定
4. 関係者に連絡 → 許可取得
5. プレスリリース起案 → レビュー → 配信
```

## PR TIMES運用

### 料金プラン（2026年時点）

| プラン | 料金 | 条件 |
|---|---|---|
| 従量課金 | 3万円/件 | 単発 |
| 月間契約 | 8万円/月 | 月30件まで |
| 半年契約 | 7.5万円/月 | 月30件まで |

**月額プランのコスト最適化**: 出せば出すほど1件あたりのコスト低下。
- 月3件: 2.67万円/件（従量より安い）
- 月8件: 1万円/件
- 月15件: 5,333円/件
- 月30件: 2,667円/件

### プレスリリース構成テンプレート

```
1. タイトル（50文字以内、具体的に）
2. サブタイトル（補足情報）
3. リード文（5W1Hを網羅、3-4文）
4. 本文
   - 背景・課題
   - 詳細内容
   - 今後の展開
5. 会社概要
6. 本件に関するお問い合わせ先
```

### 配信タイミング

| 曜日 | 効果 |
|---|---|
| 火〜木 | ◎ メディアが読む確率が高い |
| 月 | ○ 週明けで情報収集モード |
| 金 | △ 週末前で埋もれやすい |
| 土日 | × 原則避ける |

配信時刻: 10:00-11:00が最も読まれやすい

## プレスリリース → 他コンテンツ展開

1. **PR → SNS**: 配信と同時に各プラットフォームで告知
2. **PR → 記事**: プレスの背景を深掘りしたブログ記事
3. **PR → NL**: ニュースレターのトピックに含める
4. **PR → 営業資料**: 実績としてデッキに追加

## 注意点

- 関係者への事前確認は必須（特にパートナーシップ系）
- 誇大表現を避ける（「日本初」「業界唯一」は裏取り必須）
- 画像は高解像度で用意（メディアが使いやすいように）
- 問い合わせ先は必ず有効な連絡先を記載
