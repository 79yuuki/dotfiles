# 提案書テンプレート

## 用途
新規クライアントへの提案、既存クライアントへの追加提案、コンペ参加時など。

---

## スライド構成（標準10枚）

### 1. 表紙
```
[プロジェクト名] 提案書

株式会社○○御中

提案者: the company株式会社
日付: YYYY年MM月DD日
担当: [氏名]
```

### 2. 本日のアジェンダ
```
1. 現状認識・課題整理
2. 提案概要
3. ソリューション詳細
4. 導入事例・実績
5. 費用・スケジュール
6. 次のステップ
```

### 3. 現状認識（Situation + Complication）
```
■ 現状
・[クライアントの現状A]
・[クライアントの現状B]

■ 課題・問題
・[課題1]：[影響/損失]
・[課題2]：[影響/損失]

→ このまま放置すると [リスク] が発生する
```

### 4. 課題の深掘り（オプション）
```
[課題のビジュアル化]

根本原因:
  └─ 原因A
  └─ 原因B
      └─ 結果: [ビジネス影響]

数字で示す:
・[KPI現状値] → 目標 [KPI目標値]
・差分: [Gap]
```

### 5. 提案概要（Answer）
```
■ 提案タイトル
[一言でわかる提案名]

■ 3つのポイント
✓ [ポイント1]
✓ [ポイント2]
✓ [ポイント3]

■ 期待効果
・[定量効果]: [数字]
・[定性効果]: [内容]
```

### 6. ソリューション詳細
```
[アプローチ図 or フロー]

フェーズ1: [内容] ── [期間]
  └─ アウトプット: [成果物]

フェーズ2: [内容] ── [期間]
  └─ アウトプット: [成果物]

フェーズ3: [内容] ── [期間]
  └─ アウトプット: [成果物]
```

### 7. the companyについて・実績
```
■ 会社概要
・設立: [年]
・事業: [内容]

■ 関連実績
・案件A: [規模] [成果]
・案件B: [規模] [成果]
・案件C: [規模] [成果]

■ 強み
・[差別化ポイント1]
・[差別化ポイント2]
```

### 8. 費用・スケジュール
```
■ 費用概算

| フェーズ | 内容 | 費用 |
|---------|------|-----|
| フェーズ1 | [内容] | ¥[金額] |
| フェーズ2 | [内容] | ¥[金額] |
| 合計 | | ¥[合計] |

■ スケジュール（目安）
Month 1: [マイルストーン]
Month 2: [マイルストーン]
Month 3: [マイルストーン]
```

### 9. なぜ今か・なぜthe companyか
```
■ なぜ今か
・[タイミングの根拠1]
・[タイミングの根拠2]
・[競合状況 or 市場変化]

■ なぜthe companyか
・[USP1]
・[USP2]
・[リスク低減要因]
```

### 10. 次のステップ
```
■ ご確認いただきたいこと
□ 提案内容の方向性
□ 詳細ヒアリング日程
□ 予算感の確認

■ アクションプラン
→ [日付]: キックオフMTG
→ [日付]: 詳細提案書提出
→ [日付]: 契約締結目標

お問い合わせ: [メール] / [電話]
```

---

## ライティングのヒント

- **数字を必ず入れる**: 「大幅改善」→「40%コスト削減」
- **1スライド1メッセージ**: タイトルがそのまま結論になる
- **相手の言葉を使う**: ヒアリング内容をそのまま課題スライドに反映
- **CTA（行動喚起）は具体的に**: 「ご検討ください」→「来週中にお返事いただけますか」
