# Unit Economics Scorecard

## Minimum inputs
- decision unit
- expected revenue
- expected direct cost
- expected gross margin
- payment timing
- recurring fixed-cost impact
- delivery complexity
- learning value hypothesis

## Quick scorecard

### 1. Gross margin
- High: strong margin with clear delivery control
- Medium: acceptable margin but operationally noisy
- Low: weak margin or unknown delivery cost

### 2. Fixed-cost load
- High risk: adds recurring cost or permanent support burden
- Medium risk: manageable with clear owner
- Low risk: mostly variable cost

### 3. Collection quality
- Good: prepay / deposit / quick collection cycle
- Mixed: moderate delay or some working-capital strain
- Bad: long delay, front-loaded cost, weak collection leverage

### 4. Learning value
- Strong: creates reusable asset, proof point, or new route to market
- Medium: some learning but unclear reuse
- Weak: mostly bespoke, little compounding value

## Recommendation rule of thumb
- Go: margin solid, fixed-cost risk low, collection tolerable, learning real
- Revise: one of the above is weak but fixable via scope, price, or terms
- Reject: multiple weak dimensions and no credible leverage story

## Common rescue levers
- add setup fee
- reduce scope
- require deposit
- reduce payment terms
- remove bespoke support
- make learning deliverables explicit
- convert one-off work into reusable package
