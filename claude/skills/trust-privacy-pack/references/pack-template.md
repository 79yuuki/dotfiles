# Trust / Privacy Pack Template

## 1. Trust summary
- what the product does
- what trust risks matter most
- who owns the trust surface

## 2. Data categories
- account / identity
- content / payloads
- analytics / logs
- payments
- support / abuse reports

## 3. Retention matrix
For each category capture:
- purpose
- storage location
- retention period
- delete trigger
- exceptions

## 4. Public pages to draft
- privacy notice
- retention / deletion note
- safety / acceptable use note
- abuse / takedown contact
- security / trust page

## 5. Questions that require review
- jurisdiction / law-specific requirements
- contractual commitments
- security guarantees
- third-party processor promises
