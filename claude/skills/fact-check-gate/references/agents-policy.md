# AGENTS.md 反映用ポリシー

以下をAGENTS.mdの安全セクションまたは独立セクションに追加する。

---

## 🔍 ファクトチェックゲート（ハルシネーション防止）

**対象:** ドメイン・URL・価格・日付・人名・製品名を含む外部公開成果物

### 原則
1. **推測禁止カテゴリ:** domain, url, price, person, date, product は推測で埋めない
2. **知らないなら聞く:** 不明な事実は `unknown` としてuserに質問する。質問は失敗ではない
3. **facts.json 必須:** 外部公開物の生成前に facts.json を作成・確認する
4. **ファクトファースト:** レビューは文体より先にファクト突合を実施
5. **blocking未解決 = 納品禁止:** blocking: true の unknown が残る間は成果物を納品しない

### フロー
```
事実洗い出し → facts.json作成 → unknownsを質問 → 回答待ち
→ 生成（facts.jsonからのみ引用）→ 生成後ファクト突合 → 納品
```

### facts.jsonの置き場所
- プロジェクトルートに `facts.json` を配置
- 複数成果物がある場合は `facts/` ディレクトリ配下

### サブエージェントへの伝播
サブエージェントにタスクを委譲する際、facts.jsonの存在と内容を必ず伝える。
サブエージェントが新たな事実情報を必要とした場合、推測せずオーケストレーターに質問を返す。
